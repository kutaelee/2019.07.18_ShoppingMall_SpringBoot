# 2019.07.18_ShoppingMall_SpringBoot

### 주요환경
* Server
    * Springboot
    * Java
    
* Mustache   

* DB
    * Mysql
    * JDBC
    
* frontend
  * jquery
  * html
  * css
### 주요기능
* 로그인,회원가입
   * spring security
* 메인페이지
   * 주제별 리뷰 섬네일 및 미리보기
* 리뷰
   * 좋아요 기능
   * 리뷰 작성 ckEditor 적용
   * 리뷰 제조사별 금액 별 상세검색
   * AWS S3,RDS 사용
   * 메인,서브 카테고리
   * 카테고리 별 리뷰 리스트 제공
   * 리스트 페이징
